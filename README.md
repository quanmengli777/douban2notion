# 将豆瓣电影和读书同步到Notion


本项目通过Github Action每天定时同步豆瓣电影和读书到Notion。

* 豆瓣电影预览效果：https://douban-movie.malinkang.com/
* 豆瓣图书预览效果：https://douban-book.malinkang.com/
